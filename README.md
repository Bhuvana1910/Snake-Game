Snake Game (JavaScript)

A simple and interactive Snake Game built using HTML, CSS, and JavaScript.
The game includes score tracking, high score storage, timer, and restart functionality.

Features
Smooth snake movement using arrow keys
Random food generation
Score tracking system
High score saved using localStorage
Game over detection (wall + self collision)
Start and restart functionality
Timer display
Responsive grid-based board

Tech Stack
HTML
CSS
JavaScript

How to Run the Project
Clone or download the project
Open the folder
Run index.html in your browser

No installation required.

Controls
⬆️ Arrow Up → Move Up
⬇️ Arrow Down → Move Down
⬅️ Arrow Left → Move Left
➡️ Arrow Right → Move Right

Game Logic Overview
The board is divided into a grid using rows and columns
Snake moves step-by-step in a direction
Food appears randomly on the grid
When snake eats food:
Score increases by 10
Snake grows
Game ends when:
Snake hits the wall
Snake collides with itself

Storage
High score is saved using browser localStorage
